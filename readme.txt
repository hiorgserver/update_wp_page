== Description ==
Aktualisiert eine Seite durch Daten, die per API abgerufen werden.
Die Aktualisierung erfolgt bei Aufruf einer Seite, die den Shortcode [update_wp_page page_id=?] enthält
 
== Installation ==
= Voraussetzungen =
* WordPress Version 5.0 und später (getestet mit 5.0.3)
 
= Installation =
1. Plugin im Menü 'Plugins' installieren und aktivieren
1. Erstellen Sie eine neue Seite (ohne Verlinkung im Menü), welche den Shortcode [update_wp_page] enthält
1. Rufen Sie die URL dieser neuen Seite auf
